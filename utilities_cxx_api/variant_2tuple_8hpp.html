Detailed Description

Contains type traits for working with variants as if they were tuples.

Much of the template meta-programming dealing with template parameter packs is implemented in terms of std::tuple. We have need to use that functionality with other variadic types, such as std::variant. The simplest way to do this is to define routines for converting std::variant to std::tuple and vice versa. N.B. These conversions are only of the type, they will not actually convert the instances.
