Detailed Description

template<typename LHSType, typename RHSType>
class utilities::dsl::Divide< LHSType, RHSType >

Represents the division of two terms.

Template Parameters
LHSTypeThe type of the object on the left of the division sign.
RHSTypeThe type of the object on the right of the division sign.

This class is essentially a strong type over top of BinaryOp to signal that the binary operation is division (or at the least represented with a divide sign).
