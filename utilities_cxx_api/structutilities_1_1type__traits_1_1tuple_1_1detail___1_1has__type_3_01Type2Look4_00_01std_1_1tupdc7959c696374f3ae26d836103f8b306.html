template<typename Type2Look4, typename Type0, typename... OtherTypes>
struct utilities::type_traits::tuple::detail_::has_type< Type2Look4, std::tuple< Type0, OtherTypes... > >

Specializes has_type for when there are types in the std::tuple, but the 0-th type is not the type we are looking for.

This specialization is the loop body for determining if the type Type2Look4 appears in the tuple. This specialization is selected every time there is at least one type in the tuple, but the 0-th type is not the type we are looking for. This specialization inherits from the specialization of has_type for std::tuple<OtherTypes...> (i.e., the tuple without the 0-th type in the present tuple).

Eventually the class hierarchy either inherits from has_type with an empty tuple, in which case this class contains a static constexpr member value set to false, or it inherits from has_type with a std::tuple where the 0-th type is the type we are looking for. In the latter case this class contains a static constexpr member value set to true.
