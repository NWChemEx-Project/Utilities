Detailed Description

template<typename LHSType, typename RHSType>
class utilities::dsl::Add< LHSType, RHSType >

Represents the addition of two terms.

Template Parameters
LHSTypeThe type of the object on the left of the plus sign.
RHSTypeThe type of the object on the right of the plus sign.

This class is essentially a strong type over top of BinaryOp to signal that the binary operation is addition (or at the least represented with a plus sign).
